# Trabajo Práctico 1 – JavaScript
Aplicación web desarrollada con JavaScript para gestionar tareas

## Funcionalidades 

- Agregar tareas
- Eliminar tareas
- Marcar tareas como completadas
- Filtrar tareas 
- Evitar tareas duplicadas
- Persistencia de datos con localStorage

## Tecnologias utilizadas

- HTML5
- CSS
- JavaScript 
- Bootstrap 5

## Conceptos aplicados 

- Manipulación del DOM
- Eventos
- Arrays y objetos
- Funciones flecha
- localStorage
- querySelector
- addEventListener 
- JSON.stringify()
- JSON.parse()
- createElement()
- appendChild()

## Extras 

- Bootstrap
- Buscador de tareas
- Botón completar
- Botón eliminar
- Validación de duplicados
